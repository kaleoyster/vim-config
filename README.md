## Get started
Three simple steps to get started:
1. Clone the vim-config
```
 git clone https://github.com/kaleoyster/vim-config
```
2. Change directory
````
 cd vim-config 
````
3. Replace exisiting .vimrc file with vim-config .vimrc
```
 cp .vimrc ~/.
```
